# ts-unit-test-generator README

Automatically generate unit tests using OpenAI's API.
Generate a test file named *.test.ts in the same hierarchy as the target source file

- TypeScript
- ts-jest

## Extension Settings

Set your API KEY to `openai.apiKey`
